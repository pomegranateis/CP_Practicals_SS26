# Practical 3 - MST & All-Pairs Shortest Path Algorithms

Pema Tshering Yangchen — 2230295

This practical implements and analyzes:
- Borůvka's algorithm (MST)
- Floyd–Warshall (All-pairs shortest paths)
- Johnson's algorithm (All-pairs with negative weights)

Folder structure
- `P1_Boruvka/` — Borůvka implementation and trace
- `P2_FloydWarshall/` — Floyd–Warshall implementation and trace matrices
- `P3_Johnson/` — Johnson implementation, Bellman-Ford reweighting and Dijkstra runs

Run instructions and deliverables are in each subfolder.
